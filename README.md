# Cara Instalasi Program
*Pastikan service mongodb sudah dijalankan
## NPM
```js
git clone https://github.com/ikraduya/devchallenge-TwelveDays-backend.git
npm install
npm install -g nodemon
nodemon app
```
## Yarn
```js
git clone https://github.com/ikraduya/devchallenge-TwelveDays-backend.git
yarn install
yarn start
```
