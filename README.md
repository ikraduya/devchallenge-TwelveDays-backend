# Cara Instalasi Program
## NPM
```js
git clone https://github.com/ikraduya/telkom-codex-front.git
npm install
```
## Yarn
```js
git clone https://github.com/ikraduya/telkom-codex-front.git
yarn install
```

# Cara Menjalankan Program
*Pastikan service mongodb sudah dijalankan
## NPM
```js
npm start
```
## Yarn
```js
yarn start
```
